When the option OutDec is not \texttt{.}, put numbers in \texttt{\textbackslash{}text}. See \#348.

\begin{Schunk}
\begin{Sinput}
> options(OutDec = ',')
\end{Sinput}
\end{Schunk}

This is the first test. abc 0,6 def

another test $a = 0,6$.

and the last one $a = 0.6$.

\begin{Schunk}
\begin{Sinput}
> options(OutDec = '.')
\end{Sinput}
\end{Schunk}

This is the first test. abc 0.6 def

another test $a = 0.6$.

and the last one $a = 0.6$.
